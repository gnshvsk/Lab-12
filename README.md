Лабораторная работа №12 
Работа со строками

Выполнила: Гонашевская З.М., гр. ИВТ-173

Задание: создать класс MixedLetters с методом, принимающим и возвращающим текстовый аргумент.
Исходное значение - строка из нескольких слов
Выходное значение - строка из слов в том же порядке, но с перемешанными буквами кроме первой и последней в каждом слове.

Данная программа считывает строку, введенную пользователем с клавиатуры, и выводит в консоль результат -
текст с перемешанными буквами в каждом слове кроме первой и последней.

Для запуска программы необходимо открыть файл run.bat
